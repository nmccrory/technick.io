About

Technick is a full-stack blog and portfolio built by Nick McCrory. MongoDB and Node are at the core of the backend in the stack, while Angular and Skeleton CSS boilerplate make for a lithe frontend. Technick emerges from a long passion for technology and desire to learn about, share, and develop different ideas all based around engineering for the future. I will be discussing fast-growing topics in computing and technology, as well as sharing code tutorials for other aspiring developers.
